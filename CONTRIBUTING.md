# Contributing to resource-nexus

First of all, thank you for your interest in contributing to resource-nexus! Your contributions help make this project better for everyone.

By contributing to this repository, you agree to the following:

## 1. Licensing

All contributions are licensed under the MIT License.  
By contributing, you agree that your contributions will be added to resource-nexus under MIT License, and that the existing copyright notice of the repository will remain intact.

## 2. Code Standards

- Follow the existing coding style and conventions.  
- Write clear, maintainable code with comments.  
- Include tests for new features or bug fixes when applicable.
